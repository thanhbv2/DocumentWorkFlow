## Git basic
### Branch
* Tạo mới branch
>git branch {name branch}
* Chuyển sang nhánh
> git checkout {name branch}
* Tạo mới và chuyển sang nhánh mới
> git chechout -b {name branch} && 
* Tạo empty commit
> git commit --allow-empty "[start] [skip ci] {title task} {URL task}"
* Delete branch
> git branch -d {name branch}
* List all branch
> git branch
